ISO19115-1:2014 GA Profile Schema

Contents

This is the home of GA's profile of the ISO19115-1:2014 standard for geographic metadata. This profile consists of the following, human readable documents:

  1. index.html - this document, an HTML information page and the landing page of this profile. Different elements of the profile are addressed below
  2. Metadata Profile Definition Document - the formal definition document of this profile

Related to this is the GA Profile of ISO19115-3:2016, the standardised XML schema representation of ISO19115-1:2014 which contains the following resources:

  1. schema.xsd - the XML Schema Definition of the profile
  2. codelists.xml - an XML document containing all of the codelists used with the schema that GA has extended
  3. lookups.xml - an XML document containing lookup values for elements within the standard that GA prefers to constrain in user interfaces, sucha s drop-downs, rather than allow free text

The next sections in this document describe the three elements of the XML schema representation of this profile in borad outline. For further details, see the Metadata Profile Definition Document.

Code List extensions

This profile makes use of extended versions of the ISO standard's code lists. The original code lists are delivered within this standard's document and machine readable versions of those code lists are delvered in the XML schema representation of it. Here are web page versions of the extended code lists with the extensions indicated. the machine-readable versions of these are delivered in the XML schema version of this profile, as listed above.

Lookups

GA has implemented a number of lookup tables to allow catalogue tools to provided select lists for elements within the standard that are otherwise free text fields. These lookup tables are mostly taken from lookup tables provided by the GeoNetwork open source catalogue tool but we have augmented them and delivered our authoratitive versions of them as vocabularies.
