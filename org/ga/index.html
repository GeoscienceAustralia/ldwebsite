Organisations Register

About this Register

This is the static register (index) of organisations described by GA in a Semantic Web manner. Its purpose is to present a static and persistent URI to each organisation which is particularly useful for machines that wish to automatically index them, more so than than humans who can search for them manually.

This register is visible, here, in HTML for humans and also in RDF for machines:

  • RDF (turtle) version

The RDF version of this page is forumulated according to the Data Provider Register ontology which is a very simple profile of the Registry ontology.
